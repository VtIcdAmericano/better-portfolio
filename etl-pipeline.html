Retail Inventory Data Warehouse & ETL Pipeline

Stack Python · SQL · Star Schema · Tableau
Type Data Engineering · Warehouse Design

Overview

Designed and built an end-to-end ETL/ELT pipeline to transform raw grocery inventory data into a structured, analytics-ready data warehouse. The project involved schema design, data cleaning, transformation logic, and integration with Tableau for reporting and inventory optimization analysis.

Key Contributions

  • Designed a star-schema data warehouse with staging, dimension, and fact tables
  • Built Python scripts to extract, clean, and load raw inventory data into PostgreSQL
  • Developed SQL transformation logic for supplier and inventory aggregations
  • Connected the warehouse to Tableau for interactive inventory optimization dashboards
  • Documented the pipeline architecture and data flow for reproducibility
