Polytech Design & Innovation Contest

Award 🥈 2nd Place — Purdue Polytech Design & Innovation Contest
Stack Raspberry Pi · Python · Computer Vision · OpenCV
Type Hardware Prototype · Motion Analysis

Overview

Designed and programmed a Raspberry Pi-based prototype that uses camera-based motion analysis to evaluate squat form in real time. The system provides immediate feedback to users, making it a practical and accessible fitness tracking tool. The project was recognized with 2nd place at Purdue University's Polytech Design & Innovation Contest.

Key Contributions

  • Programmed real-time squat form analysis using camera-based motion tracking
  • Implemented joint angle detection logic to assess movement quality
  • Built a feedback loop to notify users of form errors in real time
  • Integrated the software with Raspberry Pi hardware for a portable device
  • Presented the prototype at Purdue's Polytech Design & Innovation Contest — awarded 2nd place
