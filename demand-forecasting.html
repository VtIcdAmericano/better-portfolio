Inventory Demand Forecasting & Reorder Optimization

Stack Python · SQL · Tableau · ML Pipelines
Type Analytics · Forecasting · Optimization

Overview

Extended the analytics-ready data warehouse built in the ETL Pipeline project to develop a scalable demand forecasting and reorder optimization system. The workflow was designed to support machine-learning-driven inventory planning and future predictive model integration, with all outputs surfaced through Tableau dashboards.

Key Contributions

  • Leveraged the star-schema warehouse as a foundation for analytical modeling
  • Built Python-based forecasting workflows for demand trend analysis
  • Designed reorder point logic using historical inventory and lead time data
  • Structured the pipeline to support future machine learning model integration
  • Connected outputs to Tableau for interactive reorder and stock-level dashboards
